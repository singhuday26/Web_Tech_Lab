# Experiment 3: Pattern Design (Chessboard/Grid)

This experiment demonstrates the use of HTML and CSS to create a visual pattern (similar to a chessboard or 4x4 grid) using alternating colors.

## Features
- Uses CSS Grid or Table layout.
- Alternating colors (e.g., Orange and darker shades) to form a pattern.

## How to Run
Open `grid.html` in a web browser.
