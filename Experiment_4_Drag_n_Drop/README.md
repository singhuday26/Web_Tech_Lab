# Experiment 4: Drag and Drop in HTML5

This experiment implements the HTML5 Drag and Drop API.

## Features
- **Draggable Element**: An orange square that can be dragged.
- **Drop Zone**: An empty box where the item can be dropped.
- **Events**: Uses `ondragstart`, `ondragover`, and `ondrop` events.

## How to Run
Open `drag_and_drop.html` in a web browser. Drag the orange box into the empty box.
