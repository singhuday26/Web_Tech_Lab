# Experiment 6: Income Tax Calculator

This web application calculates the income tax payable for FY 2025-26 under the new tax regime.

## Tax Slabs (FY 2025-26)
| Income Slab | Tax Rate |
|---|---|
| 0 - 4 Lakh | Nil |
| 4 - 8 Lakh | 5% |
| 8 - 12 Lakh | 10% |
| 12 - 16 Lakh | 15% |
| 16 - 20 Lakh | 20% |
| 20 - 24 Lakh | 25% |
| Above 24 Lakh | 30% |

## Implementation Details
- **HTML**: Provides the input form for annual income and displays the tax slabs table.
- **CSS**: Styles the form for a clean, professional look.
- **JavaScript**: Implements the logic to calculate tax based on the cumulative slab system.

## How to Run
Open `index.html` in a web browser. Enter an annual income amount and click "Calculate Tax".
