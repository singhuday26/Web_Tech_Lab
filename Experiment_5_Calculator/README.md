# Experiment 5: Calculator using JavaScript

This experiment involves designing a functional **Calculator** using HTML, CSS, and JavaScript.

## Features
- **Standard Operations**: Addition (`+`), Subtraction (`-`), Multiplication (`x`), Division (`/`).
- **Additional Functions**: Clear (`C`) and Clear Entry/Backspace (`CE`).
- **Responsive Design**: Clean grid layout that mimics a real calculator.

## Logic (JavaScript)
- Listen for button clicks.
- Append numbers/operators to a display string.
- Use `eval()` (simulated) or custom parsing to compute the result when `=` is pressed.

## How to Run
Open `index.html` in a web browser.
