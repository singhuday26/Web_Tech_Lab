# Experiment 10: Online Shopping Order Process (ES6)

This experiment simulates a basic online shopping order flow using modern JavaScript (ES6) features:

- Arrow function
- Callback function
- Promise
- Async/Await

## Files

- `index.html`: Page structure with order button and output panel.
- `style.css`: Styling for a clean demo interface.
- `script.js`: JavaScript logic for simulated order workflow.

## How It Works

1. User clicks **Place Order**.
2. Order is placed and confirmed using a callback.
3. Payment is processed asynchronously using a Promise.
4. `async/await` waits for payment completion and updates output.

## Output (Sample)

- Starting order process...
- Order placed for: Smartphone
- Order confirmed for: Smartphone
- Payment Successful
- Order will be delivered soon

## How to Run

Open `index.html` in a web browser and click **Place Order**.
