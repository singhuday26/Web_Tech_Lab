# Experiment 2: Student Registration Form

This experiment involves designing a **Student Registration Form** using HTML tags.
It collects details such as:
- Username, Password
- Date of Birth, Gender
- Address, Languages Known

## Features
- Proper form labels and input fields.
- Radio buttons for gender selection.
- Checkboxes for languages.
- Dropdown menus for Date of Birth.

## How to Run
Open `registrationform.html` in a browser.
